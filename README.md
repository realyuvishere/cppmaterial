# Commands to remember
## To compile and run
```
cd <to the directory>
make <file name without .cpp>
./<same name>
```
*OR*
```
cd <to the directory>
g++ <filename>.cpp -o <filename>.<output extension>
./<filename>.<output extension>
```
## To just compile
``` 
cd <to the directory>
make <file name without .cpp>
```
